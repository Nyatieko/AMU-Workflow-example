# AMU Workflow Example

Author: Dr. Naomi Kemunto Peter

## Project Description

This repository contains an example analytical workflow developed during my PhD research on antimicrobial use in semi-intensive poultry systems in Kenya.

The workflow demonstrates how longitudinal farm-level data were cleaned, validated, analyzed, and visualized using R and R Markdown.

## Skills Demonstrated

- Data cleaning and validation
- Longitudinal data management
- Epidemiological analysis
- Antimicrobial use quantification
- Mixed-effects regression modelling
- Data visualization
- Reproducible reporting using R Markdown

## My Role

I independently developed this analytical workflow, including data cleaning, validation, antimicrobial use quantification, statistical modelling, visualization, and reporting.
